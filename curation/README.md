# Legal-form curation — hand-off for the backend dev

These files add curated **legal-form → companyType mappings** for the jurisdictions
that currently have none. They are a **backend data drop**: the app reads them from
the document-search backend via `GET .../document-search/legal-forms?jurisdiction=`
(read-only — there is no write API). To make these ⭐ show up in the app's manual
legal-form picker, **ingest them into the same `legal-forms` store that already holds
the 14 mapped jurisdictions** (AT, BE, CH, ES, FR, GB, IE, IT, LU, NL, PL, PT, SE, US).

After ingesting `ALL.json`, **all 87 manual jurisdictions** (5 already mapped + 82 here)
have legal-form mappings — 100% manual coverage. Automatic jurisdictions (DE, HK, SG,
Cayman, BVI, Jersey, …) are intentionally excluded: their forms are inferred automatically.

## Record shape (matches the existing rows exactly)
```json
{ "jurisdiction": "MC",
  "local": "Société à Responsabilité Limitée",  // local-language name
  "abbr": "SARL",                                // abbreviation, or "—"/"" if none
  "entityType": "Company",                       // ONLY "Company" or "Partnership"
  "companyType": "Limited Liability Company" }   // MUST be a value from GET .../company-types
```
- `entityType` is constrained to **Company** | **Partnership** (partnerships = general/
  limited/limited-by-shares; everything else incl. foundations, co-ops, associations,
  trusts = Company).
- `companyType` was validated against the live `/company-types` vocabulary — **every value
  is guaranteed to exist** in it (no typos / no new values introduced).

## Confidence
- **`entityType` + `companyType` (what drives the KYC.com create) are reliable everywhere.**
- **`local` + `abbr` are best-effort for ~17 obscure territories** and should get a native
  review before or after ingest:
  `AF CC CX KG KH LA LS MM NP PF PG PS SB TO TT UZ YE`.
- Judgment calls kept (approved): LI Anstalt→Other Corporation, LI Treuunternehmen→Trust,
  MC Société Civile→General Partnership, ZA Close Corporation→Limited Liability Company,
  ZA Personal Liability Co→Incorporated, EE Osaühing→Limited Liability Company.
